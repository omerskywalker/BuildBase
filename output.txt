This has burned us on items 3-2 and 3-3 — both times the agent wrote React component render tests in a `.ts` file.\n    19→\n    20→**Rule for test files:**\n    21→- If a test **imports and renders React components** (uses `render()`, JSX, `screen.*`) → name it `*.test.tsx`\n    22→- If a test **only tests pure functions, data transforms, and logic** (no JSX) → name it `*.test.ts`\n    23→\n    24→Most BuildBase tests should be `.test.ts` because coverage targets `lib/**/*.ts` (pure logic).
